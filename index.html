FAQ

Frequently Asked Questions

01

Why are you doing a Vulnerability Scanning for free?

Many customers have mistakenly shown us reports from vulnerability scans, thinking they had undergone penetration testing. These automated scans require little effort from providers, and we offer them at no cost. Furthermore, some clients are hesitant about penetration testing due to cost concerns. They often judge the value based on the number of vulnerabilities found. However, a free vulnerability scan can be a starting point to determine if a more comprehensive penetration test is necessary.

02

What is included in web application vulnerability assessment?

The assessment of web application vulnerabilities is centered around evaluating the functionalities of the application, with a primary focus on identifying and addressing vulnerabilities at the application layer. This process is particularly concerned with mitigating risks associated with the OWASP Top Ten Web Application Security Risks.

03

What are the different types of penetration tests?

The best way to test your system's security depends on what you're trying to find. You can simulate:

Outside hackers: This "black box" test doesn't use any logins, so it won't catch weaknesses requiring access.
Malicious insiders: This "grey box" test uses logins, allowing it to find both external and internal vulnerabilities.
Full access: This "white box" test combines a regular test with a code review, looking for deep flaws.

04

What is included in the pentration tests report?

We offer a comprehensive and high-quality Penetration Testing report, ensuring accuracy by minimizing false positives. Here's what you can expect:
Clear overview: We'll start with a concise summary of the entire engagement.
Transparent scope: We'll explicitly define what was tested.
Prioritized findings: A clear list of vulnerabilities, ranked by importance.
In-depth analysis: We'll explain each vulnerability's potential impact and how to fix it.
Critical issue alerts: We'll notify you immediately of any potentially devastating vulnerabilities, allowing for prompt remediation.
In short, our actionable reports equip you with the knowledge and guidance needed to effectively address security risks.
